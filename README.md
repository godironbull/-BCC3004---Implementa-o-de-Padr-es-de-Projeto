# BCC3004 Implementacao de Padroes de Projeto
 
Este projeto reúne três padrões de projeto clássicos — um criacional, um estrutural e um comportamental — todos amplamente difundidos no desenvolvimento de software. O objetivo é demonstrar, de forma clara e prática, como esses padrões funcionam, qual problema cada um resolve e como podem ser implementados em código.

Os padrões documentados foram escolhidos a partir do catálogo do Refactoring Guru, uma das referências mais completas e organizadas sobre Design Patterns. Cada padrão possui:

📝 Explicação do seu propósito

🎯 Qual problema resolve no design de software

🧩 Solução proposta pelo padrão

📊 Diagrama UML ilustrando sua estrutura geral

💻 Implementação prática em código

📚 Referências diretas ao Refactoring Guru (por questões de direito autoral)

O projeto foi criado com foco educacional, servindo como um guia compacto, visual e didático para estudantes, profissionais e qualquer pessoa que deseje compreender e aplicar padrões de projeto no dia a dia de desenvolvimento.
